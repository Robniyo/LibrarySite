## Library CRUD Project Assignemt

## Overview
This is a PHP-based Library Management CRUD system with authentication, sessions, and cookies.  
It allows users to signup, login, manage books, and see their user identity on all pages.

---

## Features
- User signup and login
- Password hashing & verification
- Session-based access control
- CRUD operations: Create, Read, Update, Delete
- Logout functionality
- User identity displayed on all pages

---

 ## Setup Instructions
1. Import `library_db.sql` into phpMyAdmin.
2. Configure `db.php` with your database credentials.
3. Open the project in XAMPP/WAMP and run `index.php`.

---

## Authentication & Security
- PHP sessions are used for access control.
- Cookies implemented for “Remember Me” (optional).
- Passwords hashed using `password_hash()`.
- Prepared statements prevent SQL injection.
